# douban movie mini program
一个微信小程序开发豆瓣电影demo

## Getting Started
###  Installation
```
# 克隆仓库到指定的文件夹
$ git clone git@github.com:xifengzhu/douban-movie-mini-program.git

# cd douban-movie-mini-program
```
### Develop

打开微信Web开放者工具，将下载的项目添加进去，填写或选择相应信息
